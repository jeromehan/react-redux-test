# react-redux-test

## 技术栈：

react + redux + less + ES6/7 + webpack

## 下载
  git clone https://github.com/jeromehan/react-redux-test.git

 	cd react-redux-test

 	npm install
  
## 运行（nodejs 6.0+）
```
 npm run dev (正常编译模式)
 访问 http://localhost:8088
 npm run dist （发布生产版本，对代码进行混淆压缩，提取公共代码，分离css文件）
```
